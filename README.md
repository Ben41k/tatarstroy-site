# Татарстрой — статический многостраничный сайт

Сайт-визитка строительной компании «Татарстрой» на чистых HTML/CSS/JS без сборщика.

## Структура страниц

- `index.html` — главная страница с hero-секцией, превью услуг, проектов, контактов и быстрой формой заявки.
- `services.html` — карточки услуг с переходом на `contacts.html` и передачей выбранной услуги в URL (`?service=...`).
- `projects.html` — примеры выполненных проектов с изображениями из `images/`.
- `about.html` — информация о компании, подходе и преимуществах.
- `contacts.html` — контактные данные, форма обратной связи и встроенная карта.

## Переиспользуемые компоненты

Компоненты подключаются динамически через атрибут `data-include`.

- `components/header.html` — шапка сайта с навигацией и кнопкой burger-меню для компактного режима.
- `components/footer.html` — футер с контактами и быстрыми ссылками.

## Клиентская логика

Вся клиентская логика находится в `js/main.js`:

- загрузка `header`/`footer` через `fetch`;
- автоматическая подсветка текущего пункта меню (`aria-current="page"`);
- адаптивное поведение шапки: переключение в compact-режим, открытие/закрытие burger-меню, закрытие по `Esc` и по клику на ссылку;
- валидация формы обратной связи (имя, телефон, тип услуги, сообщение) с выводом ошибок;
- автоподстановка услуги в форму на `contacts.html` из query-параметра `service`.

## Стили и ресурсы

- `css/style.css` — общий файл стилей (сетка, секции, карточки, форма, адаптивность).
- `images/` — изображения проектов и графические ресурсы сайта.

## Локальный запуск

Из-за динамической подгрузки компонентов (`fetch` в `js/main.js`) страницы нужно открывать через локальный HTTP-сервер, а не через `file://`.

Пример:

```bash
python -m http.server 8000
```

После запуска откройте в браузере: `http://localhost:8000`.
